# 다함푸드 식단 관리 가이드

## 전체 흐름도

```
[1단계] 식단유형 등록 (한번만)
   ↓
[2단계] 사업장에 식단유형 연결 (한번만)
   ↓
[3단계] 식수 메뉴 등록 + 사업장에 연결 (한번만)
   ↓
[4단계] 사업장별 마감시간 설정 (한번만)
   ↓
[5단계] 매주 식단표 이미지 업로드 (매주 반복)
   ↓
[6단계] 담당자가 매일 식수 인원 입력 (매일 반복)
```

---

## 1단계: 식단유형 등록 (최초 1회)

**경로:** 메뉴 > 식단유형 관리

식단유형은 "어떤 종류의 식사를 제공하는가"를 정의합니다.

**예시:**
| 식단유형명 | 부문 | 단가 |
|-----------|------|------|
| 일반식 | 본사 | 5,500원 |
| 특식 | 본사 | 7,000원 |
| 영남일반식 | 영남지사 | 5,000원 |

**작업 방법:**
1. "식단유형 추가" 버튼 클릭
2. 유형명, 부문(본사/영남), 단가 입력
3. 저장

> 한번 등록하면 변경이 없는 한 다시 할 필요 없음

---

## 2단계: 사업장에 식단유형 연결 (최초 1회)

**경로:** 사업장 관리 > 사업장 수정

각 사업장이 어떤 식단유형을 사용하는지 연결합니다.

**작업 방법:**
1. 사업장 목록에서 해당 사업장 "수정" 클릭
2. "식단유형" 드롭다운에서 해당 사업장의 식단유형 선택
3. 저장

> 예: "삼성전자 본사" → 일반식, 특식 연결

---

## 3단계: 식수 메뉴 등록 (최초 1회)

**경로:** 메뉴 > 식수 메뉴 관리

식수 메뉴는 "담당자가 식수 인원을 입력할 때 선택하는 항목"입니다.

**예시:**
| 메뉴명 | 설명 |
|--------|------|
| A코스 | 한식 |
| B코스 | 양식 |
| 다이어트식 | 저칼로리 |

**작업 방법:**
1. "메뉴 추가" 버튼 클릭
2. 메뉴명, 설명, 순서 입력
3. 저장

**사업장에 메뉴 연결:**
- 사업장 수정 화면에서 "식수 메뉴" 항목에서 해당 사업장에서 사용할 메뉴 선택

---

## 4단계: 사업장별 마감시간 설정 (최초 1회)

**경로:** 식수 관리 > 식수 설정

각 사업장의 식수 입력 마감시간과 메뉴 개수를 설정합니다.

**설정 항목:**
| 항목 | 설명 | 예시 |
|------|------|------|
| 조식 마감 | 전날 몇 시까지 | 20:00 |
| 중식 마감 | 당일 몇 시까지 | 10:00 |
| 석식 마감 | 당일 몇 시까지 | 15:00 |
| 메뉴 개수 | 식사별 메뉴 수 (1~5개) | 중식 2개 |
| 지각 허용 | 마감 후 입력 허용 여부 | 허용/불허 |

**작업 방법:**
1. 사업장 목록에서 "설정" 클릭
2. 마감시간, 메뉴 개수 입력
3. 저장

**꿀팁:** 여러 사업장에 같은 설정을 적용하려면 → 체크박스로 여러 개 선택 후 "선택한 사업장에 설정 복제" 클릭

---

## 5단계: 매주 식단표 이미지 업로드 (매주 반복)

**경로:** 메뉴 > 주간 식단표

> 이것이 매주 반복하는 핵심 작업입니다

**작업 방법:**
1. "식단표 등록" 버튼 클릭
2. **식단유형** 선택 (예: 일반식)
3. **년도/주차** 확인 (기본값: 이번 주)
4. **식단표 이미지 업로드** (PNG, JPG)
5. 설명 입력 (선택)
6. 저장

**매주 해야 할 것:**
- 식단유형별로 1개씩 이미지 업로드
- 예: 일반식 1장, 특식 1장 = 매주 2장 업로드

**수정이 필요할 때:**
- 해당 주차의 식단표를 찾아 "수정" → 이미지 교체

---

## 6단계: 담당자가 매일 식수 인원 입력 (매일)

**경로:** 식수 관리 > 식수 인원 관리

> 이것은 각 사업장 담당자가 매일 하는 작업입니다

**작업 방법 (웹):**
1. 날짜 선택 (기본: 오늘)
2. 사업장별 식수 현황 확인
3. 입력이 필요한 사업장의 "+" 버튼 클릭
4. 식사별(조식/중식/석식) 메뉴별 인원 수 입력
5. 저장

**작업 방법 (모바일 앱):**
1. 앱 홈 > 식수 관리
2. 사업장 선택 (자동 선택됨)
3. 날짜/식사 선택
4. 메뉴별 인원 입력
5. 저장

**마감시간 주의:**
- 마감시간 전: 정상 입력
- 마감시간 후: "지각" 표시되며, 설정에 따라 입력 불가할 수 있음

---

## 한눈에 보는 관리자 체크리스트

### 최초 설정 (1회)
- [ ] 식단유형 등록 (일반식, 특식 등)
- [ ] 각 사업장에 식단유형 연결
- [ ] 식수 메뉴 등록 (A코스, B코스 등)
- [ ] 각 사업장에 식수 메뉴 연결
- [ ] 사업장별 마감시간 설정

### 매주 반복
- [ ] 주간 식단표 이미지 업로드 (식단유형별 1장)

### 매일 (담당자)
- [ ] 식수 인원 입력 (마감시간 전)

---

## 자주 묻는 질문

**Q. 식단유형과 식수 메뉴의 차이는?**
- 식단유형: 식단표 이미지와 연결되는 "식사 카테고리" (예: 일반식 5,500원)
- 식수 메뉴: 담당자가 인원을 입력할 때 선택하는 "메뉴 항목" (예: A코스, B코스)

**Q. 주간 식단표를 잘못 올렸을 때?**
- 해당 주차 식단표의 "수정" 버튼 클릭 → 이미지 교체

**Q. 사업장마다 다른 메뉴를 제공하고 싶을 때?**
- 사업장 수정 → 식수 메뉴에서 해당 사업장에 맞는 메뉴만 선택

**Q. 마감시간을 일괄 변경하고 싶을 때?**
- 식수 설정 → 체크박스로 여러 사업장 선택 → "선택한 사업장에 설정 복제"

| 식수 인원 관리 | 식수 관리 > 식수 인원 관리 |

| 사업장 수정 | 사업장 관리 > 수정 (식단유형/메뉴 연결) |
