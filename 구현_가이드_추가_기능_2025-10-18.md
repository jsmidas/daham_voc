### ⚠️ 우선순위 1: 구글 플레이 스토어 등록 준비 (필수 작업)

#### 1. 개인정보처리방침 페이지 생성 ⚠️ **필수!**
**위치**: 웹사이트에 `/privacy` 페이지 추가
**URL**: https://dahamvoc.co.kr/privacy

**필요 내용**:
- 수집하는 개인정보 항목
  - 위치 정보 (출퇴근 GPS)
  - 사진/동영상 (배식 사진)
  - 사용자 계정 정보 (이름, 이메일, 전화번호)
- 개인정보 사용 목적
- 보관 기간
- 제3자 제공 여부
- 이용자 권리
- 개인정보 보호책임자 연락처

**작업 방법**:
- 웹 프론트엔드에 `/privacy` 라우트 추가
- 또는 별도 정적 페이지 생성

#### 2. 스크린샷 캡처 (최소 2개, 권장 5개)
**필요한 스크린샷**:
- [ ] 홈 화면 - 주요 메뉴 및 출퇴근 현황
- [ ] 식수 관리 - 일별 식수 입력 화면
- [ ] VOC 작성 - 고객 의견 등록 화면
- [ ] 배식 사진 - 배식 현황 관리
- [ ] 출퇴근 - GPS 기반 근태 관리

**요구사항**:
- 크기: 1080 x 1920 (세로) 또는 1920 x 1080 (가로)
- 형식: JPEG 또는 24비트 PNG
- 최대 크기: 각 8MB

**캡처 방법**:
- 모바일 앱 실행 후 스크린샷 촬영
- 또는 에뮬레이터에서 캡처

#### 3. 앱 아이콘 512x512 준비
**현재 아이콘**: `mobile/assets/icon.png` (1024x1024)

**작업**:
- [ ] 512x512 버전으로 리사이즈
- [ ] 투명 배경 제거 (흰색 배경으로)
- [ ] PNG 형식, 32비트

#### 4. 구글 플레이 콘솔 계정 생성
**URL**: https://play.google.com/console

**필요사항**:
- [ ] 구글 계정
- [ ] 신용카드 (등록비 $25, 일회성)
- [ ] 개발자 정보 (이름/조직명, 이메일, 전화번호)

**진행 단계**:
1. Play Console 접속
2. "개발자 계정 만들기" 클릭
3. 개인/조직 선택
4. 개발자 정보 입력
5. $25 결제
6. 계정 승인 대기 (보통 즉시, 최대 48시간)

#### 5. 앱 등록 및 AAB 업로드
**준비된 파일**:
- ✅ AAB 파일: https://expo.dev/artifacts/eas/vjWryc9VXv5Hd8KyodKTqe.aab
- ✅ 앱 설명: `mobile/store_listing.md` 참고
- ✅ 등록 가이드: `mobile/play_store_guide.md` 참고

**진행 단계**:
1. [ ] Play Console에서 "앱 만들기"
2. [ ] 앱 기본 정보 입력
   - 앱 이름: 다함 VOC
   - 기본 언어: 한국어
   - 카테고리: 비즈니스
3. [ ] 스토어 등록정보 작성
   - 짧은 설명 (80자)
   - 전체 설명 (4000자)
   - 앱 아이콘 업로드
   - 스크린샷 업로드
4. [ ] 개인정보처리방침 URL 입력
5. [ ] 프로덕션 트랙에 AAB 업로드
6. [ ] 검토 제출
